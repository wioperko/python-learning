# Python Learning

## 🇵🇱 Opis

Repozytorium zawiera zbiór zadań i mini-projektów realizowanych w trakcie nauki Pythona.  
Celem jest rozwijanie umiejętności rozwiązywania problemów, pisania czytelnego kodu oraz budowania poprawnej struktury projektów.

### Cel
- utrwalenie podstaw Pythona
- rozwijanie dobrych praktyk programistycznych
- nauka modelowania problemów
- budowanie portfolio projektów

### Struktura

Każdy folder odpowiada jednemu zadaniu:

### Struktura 
01_task_name/
├── task.md        # Opis zadania
├── solution.py    # Logika biznesowa
├── run.py         # CLI runner
├── data.json      # Dane wejściowe
└── notes.md       # Wnioski i decyzje


### Założenia
- każde zadanie jest osobnym, zamkniętym problemem
- rozwiązania są proste i czytelne (bez nadmiernej komplikacji)
- w wybranych zadaniach dokumentowane są decyzje projektowe

### Status
Projekt jest w trakcie rozwoju.  
Planowane rozszerzenia:
- testy jednostkowe (pytest)
- bardziej złożone zadania

## 🇬🇧 Description

This repository contains a collection of Python exercises and small projects created during the learning process.  
The focus is on problem-solving, code readability, and building a solid foundation in software development.

### Goals
- strengthen Python fundamentals
- develop clean and readable code
- improve problem modeling skills
- build a practical coding portfolio

### Structure

Each directory represents a separate task:

01_task_name/
├── task.md        # Task description
├── solution.py    # Core logic (classes)
├── run.py         # CLI runner (argparse, JSON loading)
├── data.json      # Input data
└── notes.md       # Notes and decisions

### Approach
- each task is treated as an independent problem
- solutions prioritize clarity over complexity
- design decisions are documented when relevant

### Status
Work in progress — new tasks will be added gradually.

Planned improvements:
- unit tests (pytest)
- more advanced problems
